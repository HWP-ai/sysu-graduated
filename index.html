中山大学毕业生在 github

把信息合并进来十分简单

  1. 要求先有一个github的账号和编写html代码。
  2. 到 https://github.com/sysu/graduated 点击 fork 创建一个本仓库的拷贝到你的账号。
  3. 修改你的仓库中的 index.html 文件，添加其他相关文件。
  4. 在你的仓库的分支选择器的旁边点击 New pull request 。
  5. Base Repository 选择 sysu/graduated , base 选择 master 。
  6. 点击 Create pull request 。
  7. 在 title 中简洁地说明你的修改，并在 Write 中简单地介绍自己和所作的修改。
  8. 点击 Create pull request 。
  9. 等待有权限人士的合并。
  10. 就是这么简单！如有困难，也不要害怕搜索 github pull request 或者到 issues 留言哦。
